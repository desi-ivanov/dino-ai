# Chrome dino ai

A simple genetic algorithm for the Chrome Dino game.

## How to use

1. Open [chrome://dino/](chrome://dino/)
2. Paste the contents of [dist/main.js](dist/main.js) into the console
3. Start the game by pressing the spacebar
